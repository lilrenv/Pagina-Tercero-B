HTML

Que es html?

HTML, significa Lenguaje de Marcado de Hipertexto (HyperText Markup Language), es el lenguaje de marcado utilizado para crear y estructurar páginas web

Las páginas web están compuestas por una serie de etiquetas HTML, que son instrucciones que le dicen al navegador cómo presentar el contenido.

Igual existen mas lenguajes de Programacion

Algunos pocos de los muchos que existen son estos:

  1. JavaScript
  2. CSS
  3. Phyton
  4. SQL
  5. PHP

Las principales funciones de los lenguajes de programacion en general son:

  • Establecer qué datos debe operar un software en específico
  • Definir cómo debe almacenar los datos o trabajarlos.
  • Indicar qué acciones debe realizar de acuerdo con lo ejecutado por el usuario.
  • Permitir a los programadores comunicarse con las computadoras y crear software de manera efectiva y eficiente.
  • Escribir instrucciones que las computadoras pueden entender y ejecutar.
